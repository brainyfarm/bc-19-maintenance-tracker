# Maintenance Tracker
[![Build Status](https://travis-ci.org/azemoh/bc-19-maintenance-tracker.svg?branch=master)](https://travis-ci.org/azemoh/bc-19-maintenance-tracker)

## Requirements
  - Node.js
  - Postgresql

## To Run

 1. `npm install`
 2. `npm run db-init`
 3. `npm run db-migrate`

 4. `npm start`

 ## Testing

 `npm test`
